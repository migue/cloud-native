Cloud computing is the use of computing resources (hardware and software) that are delivered as a service over a network (typically the Internet).

Using a production database

Let´s connect our app to a production database

Create a new instance of the PostgresSQL service

cf create-service elephantsql turtle relational-database

Check our service instance has been created (note the new one has no bounded app)

cf services

name                   service       plan        bound apps      last operation
load-testing-service   blazemeter    free-tier                   create succeeded
monitoring             newrelic      standard    mimo-crud-app   create succeeded
relational-database    elephantsql   turtle                      create succeeded

Bind our new created service to our application

cf bind-service MY_APP_NAME relational-database

We need to restage our application

cf restage MY_APP_NAME

Our Cloud Foundry console has a link to our new relational database service dashboard

Monitoring our application

Let´s add monitoring support to our application

Create a new instance of the New Relic service

cf create-service newrelic standard  monitoring

Check our service instance has been created (note the new one has not bounded app)

cf services

elephantsql-d3a24               elephantsql   elephantsql   n/a       turtle     todo-list
monitoring   					newrelic      newrelic      n/a       standard   none

Bind our new created service to our application

cf bind-service MY_APP_NAME monitoring

We need to restage our application

cf restage MY_APP_NAME

Our Cloud Foundry console has a link to our new monitoring service dashboard

Doing some load testing

Let´s generate some load to our application

cf create-service blazemeter free-tier load-testing-service

Bind our new created service to our application
